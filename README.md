# Philadelphia the Global City

*Philadelphia the Global City: The Italian Legacy across Time* is an interactive map and digital collection featuring locations and photos related to the Italian legacy in Philadelphia. The website also includes original student research on each of the featured locations.

The wesbite is built using [CollectionBuilder](https://collectionbuilder.github.io/). CollectionBuilder code is licensed [MIT](https://github.com/CollectionBuilder/collectionbuilder-csv/blob/master/LICENSE). 
This license does not include external dependencies included in the `assets/lib` directory, which are covered by their individual licenses.
